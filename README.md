# ÉCLAT Nail Atelier

Адаптивный сайт студии маникюра с онлайн-записью, расписанием, двумя Telegram-ботами и модерируемыми отзывами.

## Возможности

- каталог услуг и интерактивная галерея работ;
- свободные слоты по будням с 10:00 до 20:00;
- отдельный Telegram-бот владельца для подтверждения заявок;
- отдельный клиентский бот для статусов, напоминаний и отзывов;
- напоминание о записи за 24 часа;
- Telegram Mini App с клиентским и административным режимами;
- публикация отзывов только после модерации;
- CSP, rate limiting, проверка Telegram init data и закрытая серверная конфигурация.

## Локальный запуск

1. Скопируйте `.env.example` в `.env`.
2. Заполните `BOT_TOKEN`, `CLIENT_BOT_TOKEN`, `OWNER_CHAT_ID` и `ADMIN_PASSWORD`.
3. Запустите `start-site.ps1` или `node --env-file=.env server.js`.
4. Откройте `http://localhost:4173`.

Секреты, записи клиентов и локальные журналы исключены из Git. Настройка Render описана в `DEPLOYMENT.md`.
